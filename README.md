# Portfolio Étudiant - HTML/CSS/JS

Ce projet est la base d'un portfolio d'étudiant en informatique, en **dark mode**, réalisé en HTML, CSS et JavaScript.

## Structure du projet

- `index.html` : page principale du portfolio (héros, projets, à propos, contact/footer).
- `assets/` : dossier pour les images et icônes (photo de profil, visuels de projets, etc.).
- `styles/main.css` : styles globaux, layout et responsive.
- `scripts/main.js` : interactions (scroll fluide, logique future pour les projets).

## Utilisation

1. Ouvre le projet dans ton explorateur de fichiers.
2. Double-clique sur `index.html` pour ouvrir la page dans ton navigateur.

## Où modifier / ajouter des projets

- Ajoute les images dans le dossier `assets/`.
- Ajoute/modifie les sections HTML dans `index.html` (cartes de projets, texte de présentation, etc.).
- Ajuste le style dans `styles/main.css`.
- Ajoute des interactions (par exemple, génération dynamique des projets) dans `scripts/main.js`.
